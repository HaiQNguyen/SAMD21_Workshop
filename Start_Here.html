SAMD21 Workshop

Overview

This workshop will show you the basic steps on how to set up the peripherals of the SAMD21 MCU family with Atmel Start and how to develop the applications by using Atmel Studio Within this workshop, you will have 5 labs which are dedicated to each different topic:

  • Lab 1: GPIO and UART

    This lab shows you how to set an GPIO as an input or an ouput. It also shows how to set up callback function for an external event

    You will also learn how to use UART to print debug message on the Terminal program.

  • Lab 2: Timer and PWN

    This lab shows you how to create perodics tasks by using timer and how to create a heartbeat LED with PWM

  • Lab 3: ADC and DAC

    This lab shows you how to read analog signal using an ADC and create the signal again with a DAC

  • Lab 4: "I2C"

    This lab shows you how to use I2C bus by communicating with an secure element.

  • Lab 5: SPI

    This lab shows you how to SPI bus by communicating with an Serial flash
